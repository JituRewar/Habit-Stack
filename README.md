Habit Stack: Midnight Edition
A professional-grade, persistent habit-tracking dashboard designed with a modern Cyberpunk aesthetic. This project focuses on efficient state management and high-end UI/UX principles.

Key Features
Persistent State: Uses the Local Storage API to save habits across browser sessions.

Glassmorphism UI: Advanced CSS Modules featuring backdrop-filter blurs and neon glows.

Real-time Analytics: Derived state calculations for daily progress percentages.

Interactive Celebrations: Integrated canvas-confetti for goal completion milestones.

Responsive Design: Fully optimized for mobile and desktop using CSS Grid and Flexbox.

Tech Stack
Frontend: React.js (Vite)

Styling: CSS Modules (Midnight Neon Theme)

Animations: Canvas Confetti API

Deployment: Vercel
